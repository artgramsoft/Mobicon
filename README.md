# Mobicon - 마비노기 모바일 전투 입력 매퍼

Mobicon은 마비노기 모바일(Mabinogi Mobile) 환경에서 더욱 쾌적한 전투 컨트롤을 돕기 위해 제작된 입력 매핑 도구입니다. 키보드의 특정 키를 숫자 스킬 키로 변환해주며, 마우스 클릭과 연동하여 직관적인 조작을 가능하게 합니다.

---

## 🚀 실행 방법 안내

### 방법 1: 빌드된 실행 파일(EXE) 이용 (추천)
가장 간편한 방법입니다. .NET SDK가 설치되어 있지 않아도 실행 가능합니다.

**파일 다운로드 순서:**
1.  이 페이지 우측 사이드바에 있는 **[Releases]** 항목에서 가장 위에 있는 버전을 클릭합니다.
2.  페이지 하단의 **[Assets]** 목록을 찾습니다. (만약 목록이 안 보인다면 `Assets`라는 글자를 클릭하세요.)
3.  **`Mobicon.exe`**라고 적힌 파일을 클릭하여 내 컴퓨터에 저장합니다.
4.  다운로드한 파일을 더블 클릭하여 실행합니다.

> **💡 개발자를 위한 팁:** 직접 빌드하려면 루트 폴더의 `publish.bat` 파일을 실행하세요. `./publish` 폴더에 단일 실행 파일이 생성됩니다.

### 방법 2: 소스 코드에서 직접 실행
개발 환경이 갖춰진 경우 사용합니다. (.NET 7.0 SDK 필요)

1.  터미널(또는 명령 프롬프트)을 엽니다.
2.  프로젝트 루트 폴더로 이동한 후 아래 명령어를 입력합니다.
    ```bash
    dotnet run --project Mobicon/Mobicon.csproj
    ```

## 🛡️ Windows SmartScreen 경고 해결 방법

개인 개발자가 배포하는 실행 파일은 디지털 서명이 없어 윈도우에서 "PC 보호" 경고가 뜰 수 있습니다. 이는 프로그램이 위험하다는 의미가 아니라, **"널리 알려지지 않은 파일"**이라는 의미입니다.

**실행 순서:**
1.  푸른색 **[PC 보호]** 창이 나타나면 좌측 상단의 **[추가 정보]**를 클릭합니다.
2.  우측 하단에 나타나는 **[실행]** 버튼을 클릭하여 프로그램을 시작합니다.

*본 프로그램은 오픈소스로 공개되어 있으며, 사용자의 어떠한 개인정보도 수집하지 않습니다.*

## 🎮 프로그램 사용법

### 1. 대상 확인 및 상태 표시
*   프로그램은 자동으로 `MabinogiMobile.exe` 프로세스를 감시합니다.
*   **STATUS (상태 표시줄):**
    *   `ACTIVE`: 게임 창이 활성화되어 있고 마우스가 창 안에 있을 때 (초록색)
    *   `INACTIVE`: 다른 창을 보고 있거나 마우스가 창 밖을 벗어났을 때 (회색)

### 2. 조작 키 안내
*   **전투 모드 전환 (TAB):** `TAB` 키로 매핑 기능을 켜거나(`ON`) 끌 수(`OFF`) 있습니다.
*   **일시 중지 (CTRL):** 전투 모드 중에도 `CTRL` 키를 누르는 동안에는 기능이 멈춥니다.
*   **트리거 (마우스 왼쪽 클릭):** 전투 모드가 `ON`인 상태에서 마우스 왼쪽 버튼을 **누르고 있는 동안에만** 키보드 리매핑이 작동합니다.
*   **프로그램 종료:** 콘솔 창에서 `Ctrl + C`를 누르거나, 창 우측 상단의 **[X]** 버튼을 눌러 종료합니다.

### 3. 키 매핑 리스트
| 키보드 입력 | 게임 내 출력 (숫자키) |
| :--- | :--- |
| **Q, W, E** | **4, 5, 6** |
| **A, S, D** | **1, 2, 3** |
| **Z, X, C** | **7, 8, 9** |
| **F** | **0** |

1.  **관리자 권한:** 게임이 관리자 권한으로 실행 중이면 Mobicon도 **관리자 권한**으로 실행해야 합니다.
2.  **보안:** 이 프로그램은 단순 키 변환 도구입니다. 게임 서비스 약관을 준수하시기 바랍니다.
