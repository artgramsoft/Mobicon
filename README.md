# Mobicon - 마비노기 모바일 전투 입력 매퍼

Mobicon은 마비노기 모바일(Mabinogi Mobile) 환경에서 더욱 쾌적한 전투 컨트롤을 돕기 위해 제작된 입력 매핑 도구입니다. 키보드의 특정 키를 숫자 스킬 키로 변환해주며, 마우스 클릭과 연동하여 직관적인 조작을 가능하게 합니다.

---

## 🚀 시작하기 전 준비사항 (초보자용)

Mobicon을 실행하려면 컴퓨터에 **.NET 7.0 런타임** 이상이 설치되어 있어야 합니다.

1.  **다운로드 페이지 방문:** [Microsoft .NET 다운로드](https://dotnet.microsoft.com/download/dotnet/7.0) 페이지로 이동합니다.
2.  **런타임 설치:** 'Run desktop apps' 항목에서 자신의 운영체제(Windows x64)에 맞는 **Download x64**를 클릭하여 설치 파일을 받은 후 실행합니다.
3.  **설치 확인:** 설치가 완료되면 이제 Mobicon을 실행할 준비가 되었습니다.

## 🎮 프로그램 사용법

### 1. 실행 및 대상 확인
*   `Mobicon.exe`를 실행하면 콘솔 창이 나타납니다.
*   프로그램은 자동으로 `MabinogiMobile.exe` 프로세스를 감시합니다.
*   **상태 표시줄(STATUS):**
    *   `ACTIVE`: 마비노기 모바일 창이 활성화되어 있고 마우스가 창 안에 있을 때 초록색으로 표시됩니다.
    *   `INACTIVE`: 다른 창을 보고 있거나 마우스가 창 밖을 벗어나면 회색으로 표시됩니다.

### 2. 조작 키 안내
*   **전투 모드 전환 (TAB):** `TAB` 키를 누르면 전투 모드를 켜거나(`ON`) 끌 수(`OFF`) 있습니다.
*   **일시 중지 (CTRL):** 전투 모드가 켜져 있더라도 `CTRL` 키를 누르고 있는 동안에는 리매핑 기능이 일시적으로 중단됩니다.
*   **트리거 (마우스 왼쪽 클릭):** 전투 모드가 `ON`인 상태에서 마우스 왼쪽 버튼을 **누르고 있는 동안에만** 키보드 리매핑이 작동합니다.

### 3. 키 매핑 리스트
마우스 왼쪽 버튼을 누른 상태에서 아래 키를 입력하면 대응하는 숫자키가 게임에 전달됩니다.

| 키보드 입력 | 게임 내 출력 (숫자키) |
| :--- | :--- |
| **Q, W, E** | **4, 5, 6** |
| **A, S, D** | **1, 2, 3** |
| **Z, X, C** | **7, 8, 9** |
| **F** | **0** |

## 🛠️ 주요 기능 및 안정성

*   **지능형 클릭 감지:** 게임 창 위에 다른 창(웹 브라우저 등)이 겹쳐 있는 경우, 클릭을 가로채지 않고 정상적으로 해당 창으로 포커스를 이동시켜 줍니다.
*   **키 끼임 방지:** 마우스를 떼거나 창 밖으로 나갈 때, 혹시라도 눌려 있을 수 있는 숫자키들의 '떼기(KeyUp)' 신호를 강제로 보내어 스킬이 계속 눌려 있는 현상을 방지합니다.
*   **저사양 최적화:** 활성 창 정보를 효율적으로 캐싱하여 CPU 사용량을 최소화했습니다.

## ⚠️ 주의사항

1.  **관리자 권한:** 게임이 관리자 권한으로 실행 중인 경우, Mobicon도 **관리자 권한으로 실행**해야 입력이 정상적으로 전달됩니다.
2.  **프로세스 이름:** 게임의 실행 파일 이름이 `MabinogiMobile.exe`인 경우에만 작동합니다. (버전에 따라 다를 수 있으니 확인이 필요합니다.)
3.  **안전성:** 이 프로그램은 키 입력을 단순히 변환해주는 도구입니다. 게임 서비스의 이용 약관을 준수하며 사용하시기 바랍니다.
